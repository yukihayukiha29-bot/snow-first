1. Swinging Ship

Rational Function (1/x)

Function:

y = -645/(x + 5) - 70 {-30.5 ≤ x ≤ -13} {-46 ≤ y ≤ 44}

Transformations:

  • Based on parent function y = 1/x
  • Reflection (negative direction of curve)
  • Horizontal shift 5 units left
  • Vertical stretch by factor 645
  • Vertical shift 70 units downward
  • Domain: -30.5 ≤ x ≤ -13
  • Range: -46 ≤ y ≤ 44

Transformations

To model the arc of the swinging ship, I again used a rational function because its curvature resembles the smooth circular motion of the ride. Starting with the parent function y = 1/x, I applied a negative coefficient to reverse the graph vertically so it bends in the same direction as the ship. To align the curve under the ride, I shifted the graph 5 units left using (x + 5). The coefficient 645 adjusts both the steepness and the width of the curve, allowing me to match the ship's angle precisely. Finally, I translated the entire function downward by 70 units so the highest point of the ship corresponds exactly to the highest visible point in the image.

Domain and Range

Domain: {x ∈ ℝ | -30.5 ≤ x ≤ -13}

Range: {y ∈ ℝ | -46 ≤ y ≤ 44}

The domain restriction -30.5 ≤ x ≤ -13 isolates only the arc shown in the photo. The range -46 ≤ y ≤ 44 matches the vertical movement of the ship from its lowest swing to its highest. This rational model effectively captures the dynamic curvature of the ride's motion.

2. Volcano

Rational Function (1/x²)

Function [Right]:

y = 1.1/(x + 0.2)² + 2.1 {0.65 ≤ x ≤ 2.4}

Function [Left]:

y = 1.1/x² + 2.1 {-2.7 ≤ x ≤ -0.5}

Transformations from y = 1/x²:

  • Right: Vertical stretch by factor 1.1
  • Horizontal shift 0.2 units left
  • Vertical translation 2.1 units up
  • Left: Vertical stretch by factor 1.1
  • No horizontal shift (stays at x = 0)
  • Vertical translation 2.1 units up

Transformations

To model the volcano, I chose rational functions because their steep curvature and vertical asymptotes resemble the sharp slopes of the mountain in the photo. Starting from the parent function y=1/x², I applied vertical stretches and vertical translations, and then restricted the domain and range so that only the visible parts of the volcano are shown.

For the left side, I kept the vertical asymptote at x = 0 because it lines up with the central spine of the volcano. I stretched the parent function vertically by a factor of 1.1 so that the curve becomes steeper and better matches the sharp left slope in the picture. Then I translated the graph upward by 2.1 units, which raises the whole curve until its height fits the mountain's profile. I restricted the domain to {x ∈ ℝ | -2.7 ≤ x ≤ -0.5}, so that only the portion of the curve that actually appears on the left side of the photo is included. I also used the condition y ≤ 3.8 to cut off the top of the graph at the visible summit on that side.

For the right side, I used a slightly shifted version of the same rational function. The vertical stretch upward is identical (still by 2.1 units) so the peak remains roughly as high as on the left, so the overall steepness and height of the two sides remain consistent. However, I replaced x² with (x + 0.2)², which shifts the graph 0.2 units to the left. This small horizontal shift makes the right-hand curve line up more naturally with the visible ridge and base of the volcano on that side. I limited the domain to {x ∈ ℝ | 0 ≤ x ≤ 2.4}, so that only the visible right-hand section of the volcano is drawn. I also imposed the condition y ≤ 3.6 to match the slightly lower peak height on the right side compared with the left.

Domain and Range

Domain: {x ∈ ℝ | -2.7 ≤ x ≤ -0.5} or {x ∈ ℝ | 0.65 ≤ x ≤ 2.4}

Range: {y ∈ ℝ | 2.1 < y ≤ 3.8}

From the equation and these restrictions, the corresponding ranges match the vertical height of each slope of the volcano, which corresponds to the vertical extent of each slope in the image.

3. Roller Coaster

5th-Degree Polynomial

Function:

y = -0.000014x²(x + 7)(x - 20)(x + 50) {-15.5 ≤ x ≤ 22.35}

Transformations / Structure choices:

  • Roots placed at -50, -7, 0, 20
  • Vertical scale reduced with leading coefficient -0.000014
  • Reflection across x-axis due to negative sign
  • Domain restricted to -15.5 ≤ x ≤ 22.35

Transformations

To capture the complex rises, dips, and major drop of the roller coaster, I designed a fifth-degree polynomial. Higher-degree polynomials naturally bend multiple times, which is essential for modeling the steep ascent, the small mid-coaster bump, and the dramatic downward plunge visible in the image.

I selected roots at x = -50, -7, 0, 20 so that each section of the polynomial aligns with key structural points of the real track. These roots ensure that the graph touches the horizontal axis at locations that correspond to track level points in the image. I used a very small leading coefficient, -0.000014, to control the vertical scale. Without such a small coefficient, the polynomial would grow too quickly and lose resemblance to the real ride.

The negative sign was intentional—it reflects the graph vertically so the curve slopes downward after the high peak, matching the roller coaster's sharp drop. By experimenting with coefficients and visually fitting the graph to the photo, I determined the domain -15.5 ≤ x ≤ 22.35 to isolate the visible section of the track.

Domain and Range

Domain: {x ∈ ℝ | -15.5 ≤ x ≤ 22.35}

Range: {y ∈ ℝ | -35 ≤ y ≤ 25}

This range matches everything from the bottom of the drop to the highest point of the hill, producing an accurate representation of the large-scale shape of the coaster.

4. Roller Coaster

Secant Function

Function:

y = -80 sec((π/78.5)x) + 95 {-19.7 ≤ x ≤ 19.7}

Transformations:

  • Horizontal stretch by factor 78.5/π
  • Vertical stretch by factor 80
  • Reflection across x-axis (negative sign)
  • Vertical translation 95 units upward
  • Domain: -19.7 ≤ x ≤ 19.7
  • Range: -19 ≤ y ≤ 15

Transformations

For the loop, I selected a secant function because its natural U-shaped curvature and vertical asymptotes are ideal for modeling the dramatic arch of a roller coaster loop. I began with y = sec x, then horizontally stretched it using sec((π/78.5)x), which widens the loop enough to match the proportions of the real ride.

Next, multiplying by -80 vertically stretches the graph by a factor of 80 and reflects it across the x-axis. This creates a tall upside-down arc that accurately follows the red track. Adding 95 shifts the entire curve upward so that the maximum point aligns with the highest part of the loop.

Finally, restricting the domain to -19.7 ≤ x ≤ 19.7 removes unnecessary secant branches and keeps only the visible arch in the image.

Domain and Range

Domain: {x ∈ ℝ | -19.7 ≤ x ≤ 19.7}

Range: {y ∈ ℝ | -19 ≤ y ≤ 15}

This precisely corresponds to the vertical height of the loop.
